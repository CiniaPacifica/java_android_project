### 1.서론
#### A.어플리케이션 제작동기
자바 프로그래밍 수업을 통해 익힌 코딩 기술을 활용하여 일반 택시를 이용하기 어려운 장애인분들을 위한 전용 이동 수단 서비스를 예약하고, 관리할 수 있는 어플리케이션을 제작하려고 한다.
